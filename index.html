WordPress 是什麼？

全球超過 40% 的網站都使用 WordPress 來架設，就台灣網站使用 WordPress 的人也非常多，不用擁有前後端寫程式的背景就可以入門架設，像是部落格、電商網站、一頁式網站等。

最常見的是部落客網站，早期在痞客邦或後期加入的都使用 WordPress 居多，像是痞客邦或 Facebook 等自媒體平台的廣告非常之多，市場非常競爭，但用 WordPress 架設網站的自由度很高，像是風格樣式、功能都可以客製化決定，並且對於 Google 搜尋排名非常有幫助，學習用 WordPress 架設網站絕對是斜槓性價比最高的選擇。

WordPress 架站第一步要準備什麼？

想架設網站前我們先準備「主機、網址、網頁設計」，這些費用都非常低，詳細解析在網路上有許多各式 WordPress主機推薦，等前置作業備齊就進到下一階段討論風格並實戰網頁設計，我們以部落格來說，其次形象網站設計，這兩者其實大同小異，同樣要準備以下事項。

你想架設什麼 WordPress 網站？

你對未來網站有想法了嗎？像是部落格或者公司客戶的品牌形象網站，常見 WordPress教學大多都只教學後台功能，但你想要做什麼網站？網站架構有哪些？總是一邊做一邊思考，導致最後網站無疾而終，首先需要定位你要架設網站的類型，應該需要什麼內容。

  • 部落格：常見就是想要經營個人品牌形象，各種生活瑣事都會想寫內容，舉例朝一方向像是從旅遊、美食等這些大方向延伸。基本上架構非常容易，只需要發想網站名稱、網站描述、發想撰寫大部分內容來設定分類、標籤等、聯絡我們。
  • 形象網站：這就會比較複雜，常見中小企業主必備頁面架構有關於我們、服務項目、常見問題、聯絡我們、部落格(最新消息)等來做延伸更詳細的頁面，也就是一般來說部落格更上一層想建立品牌形象要發想的事。

這也就是為什麼我建議各位新手站長先想好整體規劃，再去學習 WordPress 網站架設就會事半功倍，當都想好了遇到問題就有答案直接填入，只要分為短中長期目標就能陸續優化網站。

WordPress 網站架設流程四步驟

  • 第一步驟：購買主機、空間 (這裡買絕對值回票價)
  • 第二步驟：購買網址 ( 推薦 GoDaddy 或者 Gandi )
  • 第三步驟：文案準備 / 素材準備
  • 第四步驟：WordPress網頁設計(買版型、安裝外掛、UIUX)

網站必備文案素材有哪些？

架設網站需要準備哪些文案素材呢？你決定好網站定位走向後，下一步就要準備文案素材，當你不熟悉 WordPress 網站架設有哪些設定，又該如何準備呢？以下是基本標準要準備文案素材事項，依照網站需求來做增減即可，如果想了解詳情，請參考：經營部落格很難嗎？5個提升部落格排名方法！

  • 網站標題：整體網站名稱，有個命名公式：「網站名稱+網站定位內容+一句簡短口號」，像是快速學習網站架設能力，網站貓眾 WordPress 網頁設計！ 
  • 網站說明：描述說你的網站特色、服務內容、想要帶給使用者有什麼體驗等相關說明，必須內涵你的網站定位內容即可。
  • 文章分類：以部落格來說，就是你的文章大分類走向，依個人需求建立子分類，像是住宿、子分類就可以設定台北住宿、台中住宿，用區域+住宿，建議先想好網站起始會撰寫哪些文章即可優先設定分類。
  • 文章標籤：以部落格來說，標籤的規範就沒那麼拘謹，只需要將文章內想標記讓人查詢到即可，舉例台中旅遊必玩行程，文章內可能有逢甲夜市、宮原眼科、彩虹眷村等這些就能設定標籤，相對來說設定分類會比標籤較重要。
  • 網站選單：在網站的頂部，手機版會成三條線的漢堡選單，需要讓使用者可以更快速的了解你的網站服務內容，這依照網站定位、網站分類內容來做排序。
  • 網站LOGO：網站標示設計依照個人需求設計，目前有許多免費設計的網站可供大家去拼湊設計，這也是網站必備素材之一，新手站長們可以從短期目標設定簡易版到中長期改版網站LOGO都可以。
  • 網站風格：基本上網站風格涵蓋整體設計，建議透過標竿學習多參考其他的人網站風格、甚至使用套版都可以，搭配 Elementor 頁面編輯器可以自由設計 WordPress 網站。

WordPress 架站流程的五個重點

在我們已準備網站定位、網站文案素材後，接著就是「買網址以及主機」，但網路上主機推薦也非常多，也有大神推薦，甚至有廠商自薦自家最好用，其次我們能做的就是網址想好英文命名，這兩件事可以自行處理或者交由專業的我們來處理。

其實你只需要煩惱 WordPress 網站架設整體流程步驟有哪些？網站需要哪些內容才是重中之重，剩下主機那些就交給專業人士，省下你的時間成本，去做更有意義的事。

緊接進到下一階段就是要在 WordPress 後台將你前面準備的內容，依序設定進去即可完成最基礎的 WordPress網站，以下是主要會用到的後台功能設定。

  1. 文章 (Posts)：在[分類]、[標籤]，左側內框名稱填上你的分類，代稱建議直翻譯英文(簡短即可)。
  2. 媒體 (Media)：上傳網站LOGO等相關圖片素材。
  3. 頁面 (Pages)：通常用於聯絡我們、隱私權聲明等客製化獨立頁面，[設定]>[閱讀]的「靜態頁面」有關。
  4. 範本 (Templates)：在[主題建構器(Theme Builder)]，製作網站模板風格，此為 Elementor 頁面編輯器所用。
  5. 外觀 (Appearance)：在[選單]、[自訂]、[佈景主題]，會做到網站選單、更換網站LOGO、網站風格。
  6. 設定 (Settings)：在[一般]裡面填入網站標題、網站說明

如果你是想了解外觀佈景主題，可以參考我們寫的文章。

  • 《如何安裝WordPress主題？》
  • 《如何設定WordPress主題？》
  • 《如何移除WordPress主題？》

站長們需要知道設定完畢後，各別會對應呈現在前台網站哪裡就可以，萬一需要修改才會知道從哪裡調整。

設定 Settings

在後台登入後，左側邊欄下方有設定 Settings，點開有功能展開依序為：一般、寫作、閱讀、討論、媒體、永久連結、隱私權，你只需要依照以下設定即可。

一般 General

填上[網站標題(Site Title)] 與[網站說明(Tagline)]，再者依照使用者需求更改網站界面語言 (Site Language) 即可。

閱讀 Reading

這裡比較特別的是一般部落客設定[網站首頁顯示內容(Your homepage displays)]，點選最新文章(Your latest posts)，則前台顯示文章列表。

若[靜態頁面(A static page)]，就需要選擇靜態首頁、文章頁面(文章列表的頁面)，就需要在後台[媒體]下方頁面新增該頁面才會出現選項，此方法我會在 Elementor教學詳細說明，目前預設最新文章即可。
資訊提供中的文章顯示方式：更改為內容摘要(Excerpt)，其餘依照個人需求而定即可。

永久連結 Permalink

在[永久連結結構 (Permalink structure)]選擇文章名稱或自訂結構。

文章 Appearance

常見的 WordPress 部落格文章都在這裡新增，但文章與頁面雖然都可以新增內容，但實際用處還是不同，WordPress文章主要針對內容來做分類、標籤，讀者可以針對文章作留言回覆。

反之頁面沒有分類與標籤功能，通常用於獨立頁面，像是隱私權聲明、聯絡我們、關於我們、客製化頁面等就必須用到頁面。

  • 文章：部落格發布內容，可用於分類、標籤。
  • 頁面：用於客製化頁面內容做使用。

當網站架設前置準備，把文章內容、分類、標籤備好，建置時就可以一並新增在前台即可看到風格畫面。

外觀 Posts

我只推薦使用者兩個佈景主題選擇，因為網路上有許多教學文章分析哪一款佈景主題好用，但是有部分的新手站長們是不知道如何選擇，就跟逛街買東西一樣困難，甚至用了也無感，所以我直接推薦兩種方式。

佈景主題

  1. 購買別人的佈景主題，分為付費與免費版本。
  2. 安裝 Elementor 頁面編輯器的專屬佈景主題：Hello Elementor

第二點推薦 Hello Elementor 主題，因為預設裡面的文字大小風格全都可以在編輯器內設定，無需透過佈景主題內額外設定導致太過繁雜，當然效能方面對於一般使用上不會造成過大負擔影響。

另外安裝 Elementor 或其餘佈景主題後，點選[自訂(Customize)]後，點選網站識別，此時你準備好的網站 LOGO 上傳更換就會出現囉！

選單 Menus

網站架設前置準備選單內容此時就可以直接設定，WordPress選單的建立非常多元，你可以建立超過一個以上的選單，在不同指定的區塊做應用，選單項目可自由拖曳成下拉子選單，至於選單新增來源如下。

  • 頁面 (Pages)：已新增頁面，已發布狀態。
  • 文章 (Posts)：已新增文章，已發布狀態。
  • 自訂連結 (Custom Links)：可自由新增連結文字、網址。
  • 分類 (Categories)：來源從文章的分類來加入。

如果使用購買或免費的佈景主題，就要注意[管理選單位置]，分為頁首與頁尾兩個選單，要記得要選取選單，並儲存設定。

  • 《如何刪除WordPress選單？》
  • 《如何使用WordPress選單？五大重點整理！》

媒體 Media

在外觀的自訂上傳網站 LOGO，可先從[媒體]上傳後，在自訂網站辨識即可直接選取，像是網站內所有需要用的圖片，都必須先上傳 WordPress 才能夠做使用，建議圖片大小不能過大，否則會影響網站速度。

範本 Templates

後台有個功能[外掛(Plugins)]，點選安裝外掛，並在搜尋欄位「搜尋外掛」打上 Elementor Website Builder 即可立即安裝。

此為安裝外掛 Elementor 頁面編輯器所產生的設定，分為免費版與付費版本，在這裡我先基本介紹為什麼範本很重要，因為很多新手安裝之後卻不知道如何使用，Elementor 能做到的事非常多，可以先瞭解基本設定即可。

主題建構器

可以自由設定版面區塊內的小部件，也可以建立模板套用在全站，像是字型大小顏色、各種小部件都有，非常方便，以下像是架設部落格就會需要各別新增版型。

  • 自訂 頂部(header)
  • 自訂 底部 (footer)
  • 自訂 文章列表 (Archive)
  • 自訂 文章內頁 (Single)
  • 自訂 Error404

WordPress 教學常見問題

請問 WordPress免費嗎？

WordPress 本身是一套免費的架站軟體，你也可以使用預設的網址，通常會購買主機、網域、佈景主題或頁面編輯器等外掛，平均一年3,000-5,000元不等，依照個人需求來計算。

請問 WordPress課程值得買嗎？

不一定，建議透過網路上的 WordPress教學指南先自學，然後親自去實戰摸索，再去買課程來試試，前提是為了不白花錢，必須先有目標，也就是事前準備想好，再投入買 WordPress課程，否則你會有種被詐騙的感覺，為什麼看完還是學不會？甚至不經常練習，你就會重新歸還了！

請問 WordPress.org 和 WordPress.com 有什麼差別？

像是 WordPress.org 優勢，像是可自由挑選 WordPress 主機、可以安裝免費和付費主題還有外掛、你可以擁有所有網站資料。

WordPress.com 有如是使用某家公司平台的服務，類似使用痞客邦、Wix、Weebly 這些 網站架設平台，最大好處是它們都幫你處理好了。

但缺點是會受到很多限制，會有廣告版面的影響、不能自由安裝所有的佈景主題和外掛，而且網站所有權並不屬於你。

所以會建議使用 WordPress.org 的方式架站，也可以找 WordPress網頁設計專家。

請問 WordPress 費用怎麼算？

就購買主機一年約 6,000 – 8,000 新台幣不等，則買網址約 800 新台幣左右，其餘就是外掛等延伸功能建置費用。

請問 WordPress 外掛可以裝幾個？

這件事本身不是裝幾個問題，是安裝越多對於網站負擔越大，建議同值性的外掛擇一，否則會讓網站速度變慢，建議上網爬文確認推薦清單，同時安裝外掛也要注意最後更新時間，通常過久沒更新都會與 WordPress版本出現異常。

請問 WordPress 主機推薦哪一家？

網路上 WordPress教學大神推薦的主機可多家，但找我們處理主機，擁有超值優惠，讓你有值回票價的奢華體驗，您需要一個優質伺服器支援快速、安全的網站，提供難以置信的價值和成果，為您節省時間和金錢。

請問從痞客邦搬家到 WordPresss 怎麼辦？

若你想購買 WordPress網站架設服務，我們將免費任何託管服務主機商處將您的 WordPress 網站遷移。95% 的遷移需要不到 12 小時，歡迎聯絡網站貓眾。
